twitterのfavした画像を保存するやつ

- install

```sh
npm install
```

- setting

`config/default.yaml`

twitterのアレをソレして正しい値を入れてください

```yaml
twitter:
  consumer_key: hogehoge
  consumer_secret: hoge
  access_token_key: fugatoken
  access_token_secret: fugasecret
```


- run

```sh
npm start
```
